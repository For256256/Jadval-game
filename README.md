# 🏗️ سازه‌مارکت — پلتفرم B2B/B2C مصالح ساختمانی و صنعتی

پلتفرم استعلام و خریدِ هوشمند مصالح ساختمانی (میلگرد، تیرآهن، سیمان، گچ، لوازم برق و لوله‌کشی و…) — با داشبورد خریدار، هاب تأمین‌کننده و پنل مدیریت. ساخته‌شده با **Python (Flask)**، چندزبانه (**فارسی / عربی / انگلیسی**) با راست‌چین/چپ‌چین خودکار، و فونت اختصاصی هر زبان (وزیرمتن / Cairo / Inter).

## ویژگی‌ها
- **جست‌وجوی هوشمند RFQ با Claude AI**: کاربر نیاز خود را با متن آزاد فارسی/عربی/انگلیسی می‌نویسد (مثلاً «برای ۳۰۰ متر ساختمان در تهران، میلگرد و سیمان می‌خوام») و مدل Claude به‌صورت زنده فهرست دقیق مصالح (BOM) را با دانش مهندسی برآورد ساخت‌وساز استخراج می‌کند.
- **داشبورد خریدار (Procurement Dashboard)**: پروژه‌های فعال، فهرست مصالح، استعلام‌های باز، جدول مقایسهٔ پیشنهادها و انتخاب تأمین‌کننده.
- **هاب تأمین‌کننده (Supplier Hub)**: فید زندهٔ استعلام‌های مرتبط بر اساس شهر/دسته، ثبت پیشنهاد قیمت تک‌کلیکی، شمارش‌معکوس فرصت پاسخ‌گویی، موجودی کالا و گزارش فروش.
- **پنل مدیریت**: مدیریت کاربران و تأمین‌کنندگان، دسته‌بندی‌ها، نرخ کمیسیون و اشتراک‌ها، گزارش‌های مالی.
- **چندزبانه کامل**: تمام رابط کاربری و داده‌های نمایشی (شهر، دسته، نام تأمین‌کننده و…) به فارسی، عربی و انگلیسی با تغییر آنی زبان و جهت صفحه.
- **داده‌های واقعی**: پایگاه‌داده SQLite واقعی (نه داده‌های نمایشی)؛ تنها حساب از پیش ساخته‌شده، حساب بوت‌استرپ مدیر سیستم است.

## متغیر محیطی مورد نیاز

برای کار کردن تحلیل هوشمند RFQ باید کلید API آنتروپیک (Claude) را تنظیم کنید:

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

بدون این متغیر، باقی پلتفرم به‌طور عادی کار می‌کند اما درخواست تحلیل هوشمند با خطا مواجه می‌شود.

## اجرای محلی (برای توسعه)

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
export ANTHROPIC_API_KEY="sk-ant-..."
PORT=8080 python3 -m app.server
# سپس باز کنید: http://localhost:8080
```

## نصب تک‌خطی روی سرور اوبونتو

روی یک سرور اوبونتوی تازه (به‌عنوان root یا با `sudo`) فقط همین یک دستور را اجرا کنید؛ هیچ کلون یا کپی دستی لازم نیست:

```bash
curl -fsSL https://raw.githubusercontent.com/For256256/Jadval-game/main/install.sh | sudo bash
```

اسکریپت به‌صورت خودکار:
1. پیش‌نیازها (`python3`, `venv`, `pip`, `git`, `gunicorn`) را نصب می‌کند.
2. سورس پروژه را از همین مخزن گیت‌هاب کلون می‌کند (یا اگر قبلاً پروژه را کپی کرده‌اید و `sudo bash install.sh` را از داخل آن اجرا کنید، از همان فایل‌های محلی استفاده می‌کند).
3. محیط مجازی می‌سازد و وابستگی‌ها را نصب می‌کند.
4. سرویس `systemd` می‌سازد تا برنامه همیشه در حال اجرا و خودترمیم باشد.
5. پورت (پیش‌فرض `8080`، قابل تغییر با متغیر محیطی `PORT`) را در فایروال باز می‌کند.

برای نصب از شاخه‌ای غیر از `main`، متغیر `BRANCH` را پیش از دستور بدهید؛ مثلاً:
```bash
curl -fsSL https://raw.githubusercontent.com/For256256/Jadval-game/main/install.sh | sudo BRANCH=my-branch bash
```

برای فعال‌سازی تحلیل هوشمند RFQ، کلید Claude API خود را هنگام نصب بدهید؛ اسکریپت آن را در سرویس systemd ذخیره می‌کند:
```bash
curl -fsSL https://raw.githubusercontent.com/For256256/Jadval-game/main/install.sh | sudo ANTHROPIC_API_KEY=sk-ant-... bash
```

پس از نصب، پلتفرم در دسترس است:

```
http://<server-ip>:8080
```

## ساختار پروژه
```
sazehmarket/
├── app/
│   ├── server.py       # بک‌اند Flask، صفحات و APIها
│   ├── ai.py           # تحلیل‌گر RFQ مبتنی بر Claude API
│   ├── services.py     # منطق تجاری (پروژه‌ها، استعلام‌ها، پیشنهادها، موجودی)
│   └── data.py         # داده‌های مرجع ثابت (دسته‌ها، شهرها، کالاها)
├── templates/
│   ├── base_marketing.html / base_app.html
│   ├── index.html       # صفحهٔ اصلی
│   ├── dashboard.html    # داشبورد خریدار
│   ├── supplier.html     # هاب تأمین‌کننده
│   ├── admin.html        # پنل مدیریت
│   └── partials/
├── static/
│   ├── css/style.css
│   ├── js/ (app.js, i18n.js, home.js, dashboard.js, supplier.js, admin.js)
│   └── i18n/ (fa.json, ar.json, en.json)
├── requirements.txt
└── install.sh           # نصب‌کنندهٔ تک‌خطی
```
