# 🎮 جدولِ واژگان — بازی جدول کلمات متقاطع فارسی

یک وب‌اپلیکیشن کامل بازی جدول کلمات فارسی بر پایهٔ **پایتون (Flask)** با رابط کاربری راست‌چین، زیبا و **PWA** (قابل نصب روی موبایل).

## ویژگی‌ها
- **۱۰ مرحله** از آسان تا افسانه‌ای؛ جدول از **۷×۷** شروع شده و تا **۳۰×۳۰** بزرگ می‌شود.
- پس از کامل‌شدن هر جدول، **مرحلهٔ کشف رمز**: حروف نشان‌دار (نقطه قرمز) را کنار هم بچین و رمز پنهان را پیدا کن.
- سرنخ‌های افقی و عمودی، سیستم راهنما (آشکارسازی حرف)، امتیازدهی و نوار پیشرفت.
- پاسخ‌ها **سمت سرور** بررسی می‌شوند و هرگز در سمت کاربر لو نمی‌روند (ضد تقلب).
- بانک واژگان فارسی داخلی با سرنخ (کلمات، شهرها، مشاهیر، رنگ‌ها، افعال و…).
- رابط کاربری راست‌چین با فونت **وزیرمتن**، الهام‌گرفته از کاشی‌کاری ایرانی.

## نصب تک‌خطی روی سرور اوبونتو

پروژه را روی سرور کپی کنید و سپس:

```bash
sudo bash install.sh
```

اسکریپت به‌صورت خودکار:
1. پیش‌نیازها (`python3`, `venv`, `pip`, `gunicorn`) را نصب می‌کند.
2. محیط مجازی می‌سازد و وابستگی‌ها را نصب می‌کند.
3. سرویس `systemd` می‌سازد تا بازی همیشه در حال اجرا و خودترمیم باشد.
4. پورت را در فایروال باز می‌کند.

پس از نصب، بازی در دسترس است:

```
http://45.156.186.227:65535
```

> **نکته دربارهٔ پورت:** پورت درخواستی «۶۵۶۵۶» در پروتکل TCP نامعتبر است (بیشینهٔ مجاز **۶۵۵۳۵** است). به همین دلیل از **۶۵۵۳۵** استفاده شده است. برای تغییر، مقدار `PORT` را در ابتدای `install.sh` ویرایش کنید.

## اجرای محلی (برای توسعه)

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
PORT=8080 python3 -m app.server
# سپس باز کنید: http://localhost:8080
```

## ساختار پروژه
```
jadval/
├── app/
│   ├── server.py       # بک‌اند Flask و APIها
│   ├── crossword.py    # موتور ساخت جدول و مرحلهٔ رمز
│   └── words.py        # بانک واژگان فارسی + سرنخ
├── templates/index.html
├── static/  (app.js, sw.js, آیکون‌های PWA)
├── requirements.txt
└── install.sh          # نصب‌کنندهٔ تک‌خطی
```

## مدیریت سرویس
```bash
systemctl status jadval      # وضعیت
journalctl -u jadval -f      # لاگ زنده
systemctl restart jadval     # ری‌استارت
```
